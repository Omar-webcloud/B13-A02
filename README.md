# TechWave Podcast Website(PH-B13-A2)

A modern, responsive podcast landing page built using HTML and CSS, designed to showcase podcast episodes, highlight the host, and drive subscriptions.

---

## Live Site

 https://omar-webcloud.github.io/B13-A02/

---

## Features

### Hero Section
- Animated layout with background graphics  
- Podcast branding with title and tagline  
- Call-to-action buttons (Spotify and Subscribe)  

### About Section
- Introduction to the podcast  
- Key statistics (listeners, episodes, rating, experts)  

### Why Choose TechWave
- Premium audio quality  
- Mobile-friendly experience  
- Global community reach  
- Exclusive interviews  
- Rich learning resources  

### Featured Episodes
- Embedded YouTube podcast episodes  
- Episode titles, descriptions, and duration  

### Host Section
- Host profile with image and bio  
- Social media links  

### Footer
- Platform links (Spotify, Apple Podcasts, YouTube, Twitter)  
- Branding and copyright  

---

## Technologies Used

- HTML5 – Structure of the website  
- CSS3 – Styling and layout (external style.css)  
- Google Fonts – Outfit font  
- Font Awesome – Icons  
- YouTube Embed – Video episodes
